# 💝 İnteraktif Tebrik Kartı

Bu proje, sevdiklerinize gönderebileceğiniz 4 aşamalı, interaktif ve modern bir dijital tebrik kartıdır.

## 🚀 Özellikler
1. **Matrix Giriş:** Özel mesajın harflerinden oluşan dijital yağmur ve geri sayım.
2. **Sürpriz Mesaj:** Kelime kelime beliren mesaj ve parlayan kalp animasyonu.
3. **Tatlı GIF:** Sevimli kedi animasyonları.
4. **Scrapbook:** 3D sayfa çevirme efektli, fotoğraflı ve notlu dijital albüm.

## 🛠️ Kurulum ve Kullanım
1. Bu repoyu indirin.
2. `index.html` dosyasını tarayıcınızda açın.
3. Kendi fotoğraflarınızı `foto1.jpg`, `foto2.jpg` vb. isimlerle klasöre ekleyin.
4. `index.html` içindeki `CONFIG` kısmından mesajları kişiselleştirin.

## 🌐 Yayına Alma (GitHub Pages)
Bu projeyi ücretsiz yayınlamak için:
1. GitHub'da yeni bir repository açın.
2. Dosyaları yükleyin.
3. Settings > Pages kısmından `main` branch'ini seçip kaydedin.
4. Size verilen linki sevdiğiniz kişiye gönderin! 💝
